Module ratio

The Ratio class implements fractional numbers that can be used to represent musical quantities such as proportional metric time, durations, and just tuning intervals. Ratios can be constucted from integers, floats, or strings and are compared and combined using the standard math operators. The Ratio class inherits from RatioBase, a namedtuple superclass that declares two properties: num (numerator) and den (denominator).

def __new__(cls, num, den=None)

Constructs a Ratio from two integers, a floating point number, or a string name.

There are multiple ways to call the constructor:

  • Ratio(int, int) - creates a ratio from an integer numerator and denominator.
  • Ratio(int) - creates a ratio from an integer numerator with the denominator set to 1.
  • Ratio(float) - creates a ratio from a floating point number. To insure that the fraction will be an exact base 10 number, send the float (num) through Decimal() before calling float.as_integer_ratio() i.e.: Decimal(str(num)).as_integer_ratio()
  • Ratio(string) - creates a ratio from a string 'num/den'. Both num and den must produce valid integers.

Upon construction, the new ratio will always be expressed in its most simple form, for example Ratio(6,12) will become Ratio(1/2). (See: math.gcd()) If both the numerator and denominator are negative the ratio should be made positive by the constructor. If the numerator is positive and the denominator is negative, then normalize to a negative numerator and a positive denominator.

Parameters

num : integer | float | string
If only num is specified it must be either an integer, float, or a string containing a valid ratio expression 'a/b'. If both num and den are provided they must both be integer value.
den : int
If specified, den must be a non-zero integer denominator.

Returns

A new Ratio.

Raises

  • TypeError if the num or den is not a integer, string or float.
  • ZeroDivisionError if the denominator is 0.

Examples

  • Ratio(3,4)
  • Ratio(.5)
  • Ratio('1/3')

def compare(self, other)

Helper method that implements ratio comparison.

Returns 0 if the ratios are equal, a negative value if self is less than other and a positive value if self is GEQ other. Given two ratios the comparison is (num1 * den2) - (num2 * den1).

def tuplets(self, num, intimeof=1)

Returns a list of num sub-divisions (metric 'tuples') that sum to value of ratio * num.

Parameters

num : int
The number of tuples to return.
intimeof : int
A number that, when multiplied by the fraction itself, represents the sum of all the tuplets returned.

Returns

A list of num ratios that sum to the value of the Ratio.

Examples

  • Ratio(1,4).tuplets(3) returns three tuplets [1/12, 1/12, 1/12] which sum to Ratio(1,4).
  • Ratio(1,4).tuplets(3,2) returns three tuplets [1/6, 1/6, 1/6] which sum to ratio*2, or 1/2.
