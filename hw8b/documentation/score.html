Module score.score

A class representing a musical score.

To load a score from a MusicXml file and iterate all its objects you can do something like this:

import score  
bach = mus.import_score("bach-chorale-001.xml")  
for part in bach:  
    for staff in part:  
        for bar in staff:  
            for voice in bar:  
                for note in voice:  
                    print(part, staff, bar, voice, note)

Classes

class Score (metadata={}, parts=[])

Score.init() should initialize the score and its two attributes: self.metadata and self.parts. If parts are specified they should be added to the score by calling add_part().

Parameters

metadata : dict
A dictionary containing non-performance score properties for the score's metadata attribute. Defaults to an empty dictionary. If the score is loaded from a MusicXml file the metadata will include the following keys: 'main_key', 'main_meter', 'melodic_voices', 'static_voices', voice_ids, 'work_number', 'work_title', 'composer', 'copyright'.
parts : list
A list of score parts to initialize score the score's parts attribute. Defaults to an empty list.

Raises

A TypeError if metadata is not a dictionary.

def __str__(self)

Returns a string showing the score's title and the unique id of the instance printed in hex. To find the score title the method should check for a 'work_title' in the the score's metadata and if that does not exist it should check for a 'movement_title'. If neither metadata exists then method should return the string '(untitled)' as the title.

Examples

'<Score: "Aus meines Herzens Grunde" 0x103fa5780>'
'<Score: "(untitled)" 0x1334b57f0>'

def print_all_repr(self)

Returns a list of indented repr() strings. Every string in the list represents one repr string of a object in the score with the proper number of indents added at the beginning of that string. When later printed to the terminal (via self.print method, see below), every string in the list will be printed on its own line and indented an additional two spaces for each level.

Example

<Score: "Untitled">
  <Part: P1>
    <Staff: 1>
      <Bar: 1 Treble G-Major 4/4 STANDARD>
        <Voice: 1>
          <Note: F#4 1/4>
          <Note: B4 1/4>
          <Note: A4 1/4>
          <Note: G4 1/4>
      <Bar: 2 FINAL_DOUBLE>
        <Voice: 1>
          <Note: F#4 1/4>
          <Note: E4 1/4>
          <Note: G4 1/4>

Returns

A list of repr strings with proper indentation added.
